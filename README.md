# БалтКонт - Сайт компании

React проект для сайта компании "БалтКонт" - поставки контейнеров и комплектующих из Китая.

## Технологии

- React 18
- React Router 6
- Vite
- CSS Modules
- ESLint

## Установка и запуск

### 1. Установить зависимости

```bash
npm install
```

### 2. Запустить dev сервер

```bash
npm run dev
```

Проект запустится на `http://localhost:5173`

### 3. Сборка для production

```bash
npm run build
```

### 4. Просмотр production сборки

```bash
npm run preview
```

### 5. Линтинг

```bash
npm run lint
```

## Структура проекта

```
src/
├── components/         # Переиспользуемые компоненты
│   ├── Layout/        # Основной layout с Header и Footer
│   ├── Header/        # Шапка сайта
│   └── Footer/        # Подвал сайта
├── pages/             # Страницы
│   ├── Home/          # Главная страница
│   ├── Delivery/      # Доставка и оплата
│   ├── Catalog/       # Каталог
│   └── Contacts/      # Контакты
├── assets/            # Статические файлы (изображения)
├── App.jsx            # Основной компонент с роутингом
└── index.jsx          # Точка входа
```

## Роуты

- `/` - Главная страница
- `/delivery` - Доставка и оплата
- `/catalog` - Каталог товаров
- `/contacts` - Контакты
